NavThemes Gutenberg Block, Adds Some Standard Gutenberg Blocks to your WordPress.

== Description ==
NavThemes Gutenberg Block, Adds Some Standard Gutenberg Blocks to your WordPress. This plugin adds following Blocks to Gutenberg Editor. 

1. Team
2. Call Action
3. Heading
4. Service Blocks
5. Content Block

More Blocks Coming soon. For any feature request please mail at <a href="mailto:enquiry@navthemes.com">enquiry@navthemes.com</a> or follow us on  <a href="https://facebook.com/NavThemes">Facebook</a>  

== Installation ==
 
1. Upload `navthemes-gtblocks` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
